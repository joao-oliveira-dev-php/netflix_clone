## 🚀 Tecnologias

Este projeto foi desenvolvido com as seguintes tecnologias:


- ✔️ ReactJS

- ✔️ Context API

- ✔️ Styled Sheet

- ✔️ React-Router-Dom

- ✔️ Laravel

## ⚙ Configuração

1- Para instalar as dependências:
> npm install

2- Para iniciar a aplicação:
> npm start
